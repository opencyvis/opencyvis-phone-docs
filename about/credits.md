# Credits

OpenCyvis builds on the work of many open-source projects and contributors.

## Key Dependencies

- [Sherpa-ONNX](https://github.com/k2-fsa/sherpa-onnx) — on-device speech recognition (Apache 2.0)
- [Android Accessibility Service](https://developer.android.com/guide/topics/ui/accessibility/service) — UI element tree reading

## Inspired By

- [OpenClaw](https://github.com/openclaw/openclaw) — the open-source desktop AI agent
- [scrcpy](https://github.com/Genymobile/scrcpy) — screen mirroring and control
- [Doubao AI Phone](https://www.doubao.com) — proved that on-device AI agents are possible
