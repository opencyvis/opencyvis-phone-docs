# 参与贡献

感谢你对 OpenCyvis 的贡献意向！

## 快速开始

1. Fork 仓库
2. 克隆到你本地
3. 创建功能分支
4. 编写代码
5. 运行测试
6. 提交 Pull Request

## 构建

```bash
cd android
./gradlew assembleDebug
```

## 运行测试

```bash
cd android
./gradlew testDebugUnitTest
```

## 代码规范

- Kotlin：遵循 Kotlin 官方编码规范
- 新功能需要编写测试
- 保持提交记录聚焦且有清晰描述

## 报告问题

使用 [GitHub Issues](https://github.com/opencyvis/opencyvis-phone/issues)，请包含：
- 设备型号和 Android 版本
- 复现步骤
- 期望行为 vs 实际行为
- 如有日志请附上

## 安全问题

如发现安全漏洞，请通过邮件私下报告，不要公开发 Issue。
