# 隐私与安全

拥有完整手机访问权限的 AI 智能体，是你能运行的最高特权软件之一。这不是一个可以说「请相信我们」的地方。

## 你来选 AI 服务

OpenCyvis 不绑定模型。可以用云端服务商、私有 API、或本地模型。数据只发往你指定的地址。

## 无遥测

零追踪代码。无分析、无崩溃上报、不偷偷联网。除非你配置了 LLM Provider，App 不会连接任何服务器。

## 开源

每一行代码公开。任何人都能审计代码，验证收集了什么数据（无），确认每个权限的用途。

## 本地模型隐私

使用 Ollama 时，LLM 在你的硬件上运行。无需 API Key，无网络请求。屏幕内容不离开你的设备。
