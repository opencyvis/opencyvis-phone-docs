# Provider Profiles

Save multiple AI provider configurations and switch with one tap.

## Use Cases

- **Work**: Qwen 3.5 Plus via company API gateway
- **Home**: Local Gemma 4 via Ollama (fully private)
- **Complex tasks**: Claude Opus 4 (highest reasoning quality)

No need to re-enter API URLs and keys every time you switch. Create a profile once, then select it from the settings dropdown.

## Share a Profile

You can export any profile as a QR code (**Settings → LLM Provider → Share**). Scan it on another device to import the full configuration instantly — no typing required.
