# 虚拟显示器

大多数 AI 工具在工作时会锁定你的屏幕。OpenCyvis 在**虚拟显示器**上运行——一个隔离的后台屏幕。AI 帮你订机票的同时，你照常刷微博。

```
┌─────────────────────┐    ┌─────────────────────┐
│   你的屏幕            │    │   虚拟显示器          │
│                      │    │   (AI 在这里工作)     │
│   刷微博、聊微信、     │    │                      │
│   看视频——            │    │   订机票、发消息、     │
│   手机照常用          │    │   下单购物            │
│                      │    │                      │
└─────────────────────┘    └─────────────────────┘
      你用这个                   AI 用这个
```

## 三种状态

| 状态 | 发生什么 | 你的手机 |
|------|---------|---------|
| **聊天** | AI 在 VD 运行，你在主 App 聊天 | 完全可用 |
| **观看** | VD 画面以画中画显示 | 完全可用 |
| **接管** | VD 内容移到主屏幕显示 | 暂时占用 |

随时切换状态。觉得不对就接管，处理完交还——AI 从中断处继续。
