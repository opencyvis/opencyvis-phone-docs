# Virtual Display

Most AI tools lock your screen while they work. OpenCyvis operates on a **virtual display** — an isolated background screen. The AI books your flight while you scroll Twitter.

```
┌─────────────────────┐    ┌─────────────────────┐
│   Your screen        │    │   Virtual display    │
│                      │    │   (AI works here)    │
│   Browse, chat,      │    │                      │
│   watch videos —     │    │   Booking flights,   │
│   phone is yours     │    │   sending messages,  │
│                      │    │   placing orders     │
└─────────────────────┘    └─────────────────────┘
      You use this              AI uses this
```

## Three States

| State | What happens | Your phone |
|-------|-------------|-----------|
| **Chat** | AI runs in VD, you chat in the main app | Fully usable |
| **Watch** | VD screen shown as picture-in-picture | Fully usable |
| **Takeover** | VD content moves to your main display | Occupied (temporary) |

You can switch between states at any time. Take over if something looks wrong, hand it back when you're done — the AI resumes from where it left off.
